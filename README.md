# plg_system_menu
Megamenu - Joomla System Plugin (SPEDI srl)

## Installazione
 * Per installare scarica prima lo zip
 * Accedi al pannello di controllo di Joomla
 * Estensioni > Gestione > Installa
 * Installa il .zip
 * Ora troverai il modulo in gestione moduli
